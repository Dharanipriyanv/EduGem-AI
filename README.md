# EduGem AI 

EduGem AI is a lightweight local educational AI assistant powered by Gemma and Ollama.

## Features 

- AI-powered doubt solving
- Smart chat memory
- Voice responses
- Clean educational UI
- Local AI execution using Ollama
- Fast response generation
- Student-friendly learning assistant

## Technologies Used 

- Python (Flask)
- HTML/CSS/JavaScript
- Ollama
- Gemma 2B
- JSON storage

## How to Run 

### Start Ollama
```bash
ollama run gemma:2b
```

### Run Backend
```bash
python app.py
```
### Open Frontend
Open `frontend/index.html` in browser.
